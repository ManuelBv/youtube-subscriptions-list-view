# Public Folder

Place your YouTube subscriptions page files here for local testing:

1. **Save the HTML**
   - Navigate to https://www.youtube.com/feed/subscriptions
   - Right-click → Save As → Complete Webpage
   - Save as `youtube-subscriptions.html` (or any name you prefer)

2. **Include CSS files**
   - The saved webpage should include associated CSS files
   - Make sure all assets are properly saved

3. **Test the bookmarklet**
   - Open the saved HTML file in your browser
   - Open browser console
   - Paste the bookmarklet code to test

**Note:** This folder is gitignored to avoid committing YouTube's copyrighted content.
